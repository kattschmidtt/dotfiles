# Files that start with dots usually

Repo of configs for different coding tools (editors, terminal emulators, etc...)

## Steps to implement

Clone the project

```bash
  cd ~/
  mkdir -p ~/.config
  git clone https://github.com/kattschmidtt/dotfiles.git
```

Copy files from dotfiles folder to .config

```bash
  cp -r dotfiles/* ~/.config
```
